提交答题

基本信息

Path： /ks/20180915/questions/answer

Method： POST

接口描述：

提交答题接口

返回的错误信息场景描述：
1. 请求头token验证未通过
{
    error: 10001,
    msg: '没有访问权限'
}

2. 当请求参数格式不正确或者必填参数为空时
{
    error: 10002,
    msg: '请求参数格式错误'
}

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/json 是
token 14052ae98d89b4570b8f8842c17fa971a1b844e7 是 校验token

Body

名称 类型 是否必须 默认值 备注 其他信息
appId string 必须 应用ID,服务端根据appId作为关键索引，相同的appId,新提交的内容会覆盖原来已提交的内容
nickname string 必须 昵称
gender number 必须 性别，1男2女
answers string [] 必须 提交的答题记录

item 类型: string

返回数据

名称 类型 是否必须 默认值 备注 其他信息
status number 必须 请求的响应结果，1代表成功
body object 必须 响应的参数体
├─ type number 必须 根据答题内容，反馈的判断性格类型标识，1-9
├─ descript string 必须 性格类型描述，如“完美型”
├─ weburl string 必须 性格详情对应的h5页面url地址
├─ error object 非必须 错误信息对象，当status=0时返回
├─ code number 必须 错误代码
├─ msg string 必须 错误描述信息

获取全部答题结果列表信息

基本信息

Path： /ks/20180915/result/list

Method： GET

接口描述：

获取包含参与答题的所有人员的结果信息列表

返回的错误信息场景描述：
1. 请求头token验证未通过
{
    error: 10001,
    msg: '没有访问权限'
}
2. 当请求参数格式不正确或者必填参数为空时
{
    error: 10002,
    msg: '请求参数格式错误'
}

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
token 14052ae98d89b4570b8f8842c17fa971a1b844e7 是 校验token

Query

参数名称 是否必须 示例 备注
index 是 0 请求获取的起始数据序号，其实序号0
count 是 10 客户端请求服务端一次返回的数据条数

返回数据

名称 类型 是否必须 默认值 备注 其他信息
status number 必须 请求的响应状态，1代表请求成功，0代表请求失败
body object 必须 响应参数体
├─ list object [] 必须 列表数组

item 类型: object
├─ nickname string 必须 答题人昵称
├─ gender number 必须 性别，1代表男性2代表女性
├─ character string 必须 性格简称，如“完美”、“助人”、“和平”
├─ type number 必须 性格类型（1-9）
├─ weburl string 必须 个性详情对应的h5页面url地址
├─ totalCount number 必须 列表数据总条数
├─ error object 非必须 错误对象，当请求错误时返回
├─ code number 非必须 错误代码
├─ msg string 非必须 错误描述
